# 🔥 Forest Fire Detection using CNN

A deep learning project that detects forest fires from images using a Convolutional Neural Network (CNN) and PyTorch.

---

## 📌 Features

- CNN-based image classification
- Fire vs No Fire detection
- PyTorch implementation
- Data augmentation
- Model training and evaluation

---

## 📂 Dataset Structure

```bash
Dataset/
├── fire/
└── nofire/
```

---

## ⚙️ Installation

Install required libraries:

```bash
pip install torch torchvision matplotlib numpy pandas
```

---

## 🚀 Run the Project

Open the notebook:

```bash
forest_fire_detection_cnn.ipynb
```

Run all cells to train and test the model.

---

## 🧠 Technologies Used

- Python
- PyTorch
- CNN
- torchvision
- NumPy
- Matplotlib

---

## 📊 Project Goal

The objective of this project is to classify forest images into:

- 🔥 Fire
- 🌲 No Fire

using deep learning techniques.
